# WW2 Germany Aircraft Support System

## 完整的二战德国军事飞机模组

完整的第二次世界大战德国军事模组包，包含运输、轰炸和扫射三大系统。

## 模组功能

### 1. ME323 运输机系统
- **ME323 巨人（货运版）**：运输囚犯（128人）+ 物资（12吨）
- **ME323 巨人（兵力版）**：紧急空投精英士兵（120人）
- 平时物资补给 / 囚犯转移，紧急时刻快速部署士兵

### 2. 空中支援系统
- **Ju-87 斯图卡**：俯冲轰炸定点投弹，高伤害+起火
- **Me262 喷气战机**：高速扫射长距离压制
- 处理监狱暴乱的最终手段

### 3. 核心特性
- ✅ 不降落机制：飞机执行任务后直接离场
- ✅ 手动召唤系统：点击指挥台选择目标
- ✅ 冷却管理：防止滥用
- ✅ 中文+德文完整支持
- ✅ 历史准确：参考真实WW2数据

## 安装方法

1. 下载仓库或克隆到本地
2. 复制整个 `WW2_PrisonArchitect_Mods` 文件夹到 Prison Architect 的 mods 目录：
   ```
   C:\Users\[用户名]\AppData\Local\Introversion\Prison Architect\mods\
   ```
3. 游戏内启用模组
4. 重新开始游戏

## 使用方法

### ME323 运输机
- 建造 **ME323 货运停机坪** → 自动运囚犯和物资
- 建造 **ME323 军用停机坪** → 点击呼叫空投士兵

### 空中支援
- 建造 **空袭指挥台**
- 右键点击 → 选择"召唤斯图卡"或"召唤Me262"
- 点击地图选择目标位置
- 飞机自动执行任务后离场

## 冷却时间
- 斯图卡：90秒
- Me262：60秒
- ME323 兵力运输：120秒

## 机制详解

### ME323 货运版
- **运输容量**：128名囚犯 或 12吨物资
- **速度**：2.2（中等，符合历史沉重感）
- **功能**：平时用于日常囚犯转移和物资补给
- **成本**：8000费用，300木材，100钢铁

### ME323 兵力版
- **运输容量**：120名精英士兵（一次空投）
- **速度**：2.0（较慢，强调运输重量）
- **功能**：紧急情况下快速部署精英部队
- **成本**：15000费用，400木材，200钢铁，50弹药

### Ju-87 斯图卡
- **轰炸伤害**：85
- **范围**：3格半径
- **特效**：会引发火灾
- **速度**：3.5（中等）
- **冷却**：90秒
- **用途**：定点打击暴乱聚集区

### Me-262 喷气战机
- **扫射伤害**：35（范围伤害）
- **范围**：8格直线
- **特效**：压制囚犯
- **速度**：6.0（极快）
- **冷却**：60秒
- **用途**：大面积暴乱压制

## 历史背景

### ME323 巨人（Messerschmitt Me 323）
- 二战德国最大的货运飞机
- 真实运载量：120-130人 / 10-12吨货物
- 实战案例：北非战线物资运输、东线兵力部署

### Ju-87 斯图卡（Junkers Ju 87 Stuka）
- 著名的俯冲轰炸机
- 标志性的尖锐啸叫声（心理作用）
- 精准性强，常用于定点打击

### Me-262 喷气战机（Messerschmitt Me 262）
- 世界上第一种投入实战的喷气式战斗机
- 速度极快（900km/h）
- 装备4×30mm机炮，火力强大
